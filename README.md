# ai-english-bot
AI bot for learning English (Python, LLM)
# AI English Learning Bot

# Описание
Консольное приложение на Python для изучения английского языка с использованием локальной LLM-модели (llama3).  
Приложение помогает пользователю изучать лексику, тренировать грамматику и практиковать диалог.

---

# Функциональность

- Перевод слов (локальный словарь + AI при необходимости)
- Генерация слов по заданной теме
- Проверка знаний пользователя (интерактивный режим)
- Генерация грамматических заданий
- Диалог с AI с исправлением ошибок

---

# Использование AI

Проект использует локальную LLM (llama3) через HTTP API для:
- генерации заданий
- проверки ответов
- ведения диалога с пользователем

---

# Технологии

- Python
- requests
- REST API
- LLM (llama3)
